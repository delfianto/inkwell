# Inkwell documentation

- **[Interface guide](./interface.md)** — a walkthrough of every UI surface: the
  Scenes List/Cards/Outline views, scene status colours, the Project tab, the eBook
  metadata modal, creating a project, and the full-pane Compile builder.
- **[The UI redesign](./ui-redesign.md)** — the before/after story. Inkwell `2.0.0`
  reimplemented essentially every writing surface it inherited from Longform; this
  page pairs each old surface with its replacement and links the audit and mockups
  that drove the work.

For installation, the `Index.md` frontmatter schema, migrating from Longform, and
the build/dev setup, see the [main README](../README.md).

Design reference lives in [`ui-audit/`](ui-audit): the original usability
[audit](ui-audit/README.md) with "before" screenshots, and the
[mockups](ui-audit/mockups/MOCKUPS.md) that set the direction.
