# Word Counts

Inkwell tracks word counts per project and per scene and displays them in the **Word Count** section of the Project tab. Multi-scene projects show both the focused scene and the project total; single-scene projects show only the project total. For performance reasons, counts are updated on a delay after edits.

This fork intentionally omits the writing-goal and status-bar word-count features from upstream Inkwell. Obsidian's built-in word-count statistic covers the global status-bar use case.
